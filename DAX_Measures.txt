# DAX Measures

---

##  Sales & Profit Measures

```dax
Total Sales = 
SUM(DataCoSupplyChainDataset_clean_processing[Sales])

Total Profit = 
SUM(DataCoSupplyChainDataset_clean_processing[Net_Profit])

Profit Margin % = 
DIVIDE([Total Profit], [Total Sales], 0)
```

---

## Order Measures

```dax
Total Orders = 
COUNTROWS(DataCoSupplyChainDataset_clean_processing)

% of Total Orders = 
DIVIDE(
    COUNTROWS(DataCoSupplyChainDataset_clean_processing),
    CALCULATE(
        COUNTROWS(DataCoSupplyChainDataset_clean_processing),
        ALL(DataCoSupplyChainDataset_clean_processing)
    ),
    0
)
```

---

## Delivery & Logistics Measures

```dax
On Time Orders = 
CALCULATE(
    COUNTROWS(DataCoSupplyChainDataset_clean_processing),
    DataCoSupplyChainDataset_clean_processing[Delivery_Status] = "Shipping on time"
)

On Time Rate % = 
DIVIDE([On Time Orders], [Total Orders], 0)

Late Orders = 
CALCULATE(
    [Total Orders],
    DataCoSupplyChainDataset_clean_processing[Delivery_Status] = "Late"
)

Late Order Rate % = 
DIVIDE([Late Orders], [Total Orders], 0)

Early Orders = 
CALCULATE(
    COUNTROWS(DataCoSupplyChainDataset_clean_processing),
    DataCoSupplyChainDataset_clean_processing[Delivery_Status] = "Early"
)

Avg Shipping Days = 
AVERAGE(DataCoSupplyChainDataset_clean_processing[Days for shipping (real)])

Revenue at Risk = 
CALCULATE(
    [Total Sales],
    DataCoSupplyChainDataset_clean_processing[Delivery_Status] = "Late"
)

Revenue Lost to Late Orders = 
CALCULATE(
    [Total Sales],
    DataCoSupplyChainDataset_clean_processing[Delivery_Status] = "Late"
)
```

## Fraud & Risk Measures

```dax
Fraud Orders = 
CALCULATE(
    COUNTROWS(DataCoSupplyChainDataset_clean_processing),
    DataCoSupplyChainDataset_clean_processing[Order Status] = "Suspected Fraud"
)

Fraud Rate % = 
DIVIDE(
    CALCULATE(
        COUNTROWS(DataCoSupplyChainDataset_clean_processing),
        DataCoSupplyChainDataset_clean_processing[Order Status] = "Suspected Fraud"
    ),
    COUNTROWS(DataCoSupplyChainDataset_clean_processing),
    0
)

Revenue Lost to Fraud = 
CALCULATE(
    [Total Sales],
    'DataCoSupplyChainDataset_clean_processing'[Order Status] = "Suspected Fraud"
)

## Discount & Margin Measures

```dax
Avg Discount Rate = 
AVERAGE(DataCoSupplyChainDataset_clean_processing[Discount_Rate])

Profit Lost to Discount = 
SUMX(
    'DataCoSupplyChainDataset_clean_processing',
    'DataCoSupplyChainDataset_clean_processing'[Sales] *
    'DataCoSupplyChainDataset_clean_processing'[Discount_Rate]
)

Discount to Profit Ratio = 
DIVIDE([Profit Lost to Discount], [Total Profit], 0)

Margin vs Average = 
VAR _DeptMargin = [Profit Margin %]
VAR _AvgMargin = 
    CALCULATE(
        [Profit Margin %],
        ALL('DataCoSupplyChainDataset_clean_processing'[Department])
    )
VAR _Diff = _DeptMargin - _AvgMargin
RETURN
    _Diff

Most Discounted Dept = 
FIRSTNONBLANK(
    TOPN(
        1,
        VALUES('DataCoSupplyChainDataset_clean_processing'[Department]),
        [Avg Discount Rate],
        DESC
    ),
    1
)
```

## "Last Year" Comparison Measures

These return a descriptive text label (e.g. "vs $1.3M Last Year") and are designed to display blank (`""`) when no single year is selected via the Year slicer, rather than showing `0` or `--`.

    CALCULATE(
        [Total Profit],

RETURN
    IF(
        NOT HASONEVALUE('Date Table'[Year]),

            ISBLANK(_LY),

## "% Change" Measures (YoY Trend Indicators)

These return a formatted text string with a directional arrow (▲/▼) showing the year-over-year percentage change. By design, these display the full-period comparison regardless of slicer selection.

```dax

VAR _CM = [Total Sales]
VAR _PM =
    CALCULATE(
        [Total Sales],
        DATEADD('Date Table'[Date], -1, YEAR)
    )
VAR _perc = DIVIDE(_CM - _PM, _PM)
VAR _percformat = FORMAT(_perc, "+0.0%;-0.0%;0.0%")
RETURN
    IF(
        _perc > 0,
        UNICHAR(129157) & " " & _percformat,
        UNICHAR(129158) & " " & _percformat
    )

## Conditional Formatting (CF) Measures

These return a color name value used to drive conditional formatting on KPI cards based on year-over-year trend direction.

### Standard logic — increase = improvement (Green when up)

```dax
CF Sales = 
VAR _CM = [Total Sales]
VAR _PM =
    CALCULATE(
        [Total Sales],
        DATEADD('Date Table'[Date], -1, YEAR)
    )
VAR _perc = DIVIDE(_CM - _PM, _PM)
RETURN
    SWITCH(
        TRUE(),
        _perc > 0, "Green",
        _perc < 0, "Red",
        "Grey"
    )

CF Profit = 
VAR _CM = [Total Profit]
VAR _PM =
    CALCULATE(
        [Total Profit],
        DATEADD('Date Table'[Date], -1, YEAR)
    )
VAR _perc = DIVIDE(_CM - _PM, _PM)
RETURN
    SWITCH(
        TRUE(),
        _perc > 0, "Green",
        _perc < 0, "Red",
        "Grey"
    )

CF Profit Margin = 
VAR _CM = [Profit Margin %]
VAR _PM =
    CALCULATE(
        [Profit Margin %],
        DATEADD('Date Table'[Date], -1, YEAR)
    )
VAR _perc = DIVIDE(_CM - _PM, _PM)
RETURN
    SWITCH(
        TRUE(),
        _perc > 0, "Green",
        _perc < 0, "Red",
        "Grey"
    )

CF On Time Rate = 
VAR _CM = [On Time Rate]
VAR _PM =
    CALCULATE(
        [On Time Rate],
        DATEADD('Date Table'[Date], -1, YEAR)
    )
VAR _perc = DIVIDE(_CM - _PM, _PM)
RETURN
    SWITCH(
        TRUE(),
        _perc > 0, "Green",
        _perc < 0, "Red",
        "Grey"
    )

CF Discount Rate = 
VAR _CM = [Avg Discount Rate]
VAR _PM =
    CALCULATE(
        [Avg Discount Rate],
        DATEADD('Date Table'[Date], -1, YEAR)
    )
VAR _perc = DIVIDE(_CM - _PM, _PM)
RETURN
    SWITCH(
        TRUE(),
        _perc > 0, "Green",
        _perc < 0, "Red",
        "Grey"
    )

CF Margin vs Avg = 
VAR _Diff = [Margin vs Average]
RETURN
    SWITCH(
        TRUE(),
        _Diff > 0, "#4D7DAA",
        _Diff < 0, "#A0D1FF",
        "#888780"
    )
```
